FAQ

Pertanyaan Umum

Apa bedanya VPS dengan shared hosting?
VPS memberikan resource yang dedicated (tidak dibagi pengguna lain), full root access untuk konfigurasi bebas, dan performa yang jauh lebih stabil dibanding shared hosting.
Kenapa pakai Hetzner?
Hetzner adalah penyedia infrastruktur cloud terpercaya dengan AMD EPYC processor tercepat di dunia. Kami pilih Hetzner karena performa CPU single-core terbaik dan jaringan Singapore yang stabil.
Bedanya Genoa dan Milan?
Genoa = CPU lebih baru, clock lebih tinggi per core — cocok untuk single-thread. Milan = CPU lebih lawas tapi lebih efisien multi-thread — cocok untuk workload berat. Pilih berdasarkan kebutuhan spesifik kamu.
Itu Private Node bedanya apa?
Private Node adalah server dedicated untuk penggunaan intensif (MC besar, banyak bot). Resource lebih stabil karena tidak ada tetangga, cocok untuk performa maksimal. Tidak ada root access, dikelola oleh tim kami.
Bisa bayar pakai apa saja?
Kami menerima pembayaran via QRIS, GoPay, OVO, Dana, ShopeePay, transfer bank (BCA, BRI, Mandiri, BNI), dan pulsa (dengan rate tertentu).
Apakah bisa upgrade paket?
Ya! Hubungi support kami, sebutkan paket yang ingin dinaikkan, proses dilakukan tanpa downtime. Selisih biaya dihitung prorata sesuai sisa masa aktif paket.

Bingung Pilih Paket?

Hubungi kami sekarang dan tim kami akan membantu memilihkan paket yang paling sesuai kebutuhanmu.
